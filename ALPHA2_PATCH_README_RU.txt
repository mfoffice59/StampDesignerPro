Stamp Designer Pro 2.0 Alpha 2 Patch

Исправлено:
- удалено свойство ColumnSpacing из MainWindow.axaml;
- заменено на Margin, совместимый с GitHub Actions и Avalonia 11.1.4;

Как загрузить:
1. Распаковать ZIP.
2. На GitHub нажать Add file - Upload files.
3. Перетащить папку src целиком.
4. GitHub должен заменить существующие файлы.
5. Commit changes.
6. Перейти в Actions и дождаться новой сборки.
