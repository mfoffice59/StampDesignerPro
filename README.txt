Stamp Designer Windows App Source v1

Это проект настоящего Windows-приложения на C# WinForms.

Что есть:
- полноценное окно Windows;
- генерация круглого штампа;
- шаблоны Узбекистана и РФ;
- стиль двойной линии;
- стиль широкой синей внешней полосы с белым текстом;
- загрузка логотипа;
- перемещение логотипа мышью;
- колесико мыши меняет размер логотипа или ластика;
- мягкий регулируемый ластик;
- сила стирания;
- Undo и Redo;
- экспорт PNG 3000 x 3000;
- сохранение и открытие проектов .stamp.

Как собрать EXE:
1. Установить .NET 8 SDK от Microsoft.
2. Распаковать папку.
3. Запустить build-win-x64.bat.
4. Готовый файл будет здесь:
   bin\Release\net8.0-windows\win-x64\publish\StampDesigner.exe
