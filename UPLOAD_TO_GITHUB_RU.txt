КАК ЗАГРУЗИТЬ В GITHUB

1. Распакуйте архив.
2. Откройте репозиторий StampDesignerPro на GitHub.
3. Нажмите Add file - Upload files.
4. Перетащите все содержимое папки StampDesignerPro_2_0_Alpha1.
5. Нажмите Commit changes.

Важно:
Папка .github может быть скрытой.
Если она не загрузится через браузер:
- создайте файл вручную:
  .github/workflows/build-release.yml
- вставьте содержимое из локального файла build-release.yml.

После загрузки:
GitHub - Actions - Build Stamp Designer Pro - Run workflow.
