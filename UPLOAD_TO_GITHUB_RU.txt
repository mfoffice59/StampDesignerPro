КАК ЗАГРУЗИТЬ ALPHA 2 FULL

1. Распакуйте ZIP.
2. Откройте папку StampDesignerPro_2_0_Alpha2.
3. На GitHub нажмите Add file - Upload files.
4. Загрузите содержимое этой папки поверх текущего репозитория.
5. Commit changes.
6. Перейдите во вкладку Actions и дождитесь новой сборки.

Важно:
- Если .github скрыта на macOS, не страшно: workflow уже создан в репозитории.
- Главное обновить папку src и файлы README.md / CHANGELOG.md.
