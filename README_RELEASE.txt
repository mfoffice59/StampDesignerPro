Stamp Designer Windows 1.0 - portable release

Состав:
- StampDesigner.exe - готовая программа
- README.txt - краткая инструкция

Как пользоваться:
1. Запустите StampDesigner.exe двойным щелчком.
2. Выберите шаблон.
3. Введите текст.
4. Загрузите логотип при необходимости.
5. Перемещайте логотип мышью.
6. Включите "Ластик" для ручной коррекции.
7. Сохраните PNG или проект .stamp.

Форматы:
- PNG - готовая картинка штампа.
- .stamp - проект, который можно открыть и продолжить редактировать.

Важно:
Если Windows SmartScreen покажет предупреждение, нажмите "Подробнее" - "Выполнить в любом случае".
Это возможно потому, что приложение пока не подписано цифровым сертификатом.
